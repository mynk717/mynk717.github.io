SEO Case Study

Shifted a branded-heavy Shopify store toward non-branded discovery — 400+ new query entries in GSC footprint within 30 days.

Client — Shree Shivam Attire · shreeshivam.com · Shopify
Starting from a store where over 95% of traffic was branded search, we built non-branded discovery from scratch. Through technical fixes, canonical resolution, and targeted blog content, we grew the non-branded GSC footprint to 400+ indexed queries and drove 1,504 impressions from 6 newly published blog URLs — in under 30 days.

The Problem
  • 95%+ of all traffic was branded-only (zero discovery traffic)
  • Indexed pages dropped 40% — from 655 to 389 pages
  • Impressions declining from 4,900/day to ~2,600/day
  • Kurti-set collection: 77 SKUs, only 15 clicks/month
  • 301 redirect loop on primary collection URL
The Stack Used
  • GSC API · GA4 · Shopify GraphQL API
  • Notion API — keyword tracking + content pipeline
  • Custom Streamlit dashboard (shree_seo_engine)
  • DataForSEO · Python automation · Google Data Studio
  • GCP Service Account for API auth
What I Did
  • Resolved 301 redirect loop — fixed canonical chain on kurti-set URL
  • Sitemap resubmitted Apr 13 2026 — used as official sprint baseline
  • Published 4 blog articles targeting non-branded discovery queries
  • Built 17-issue technical register with full Notion SEO tracker
  • Automated weekly GSC reporting via Python + Streamlit dashboard
  • Set up Google Data Studio live connector via Apps Script
Results & Context
  • Result: Grew non-branded keyword footprint from near-zero to 400+ queries in GSC — a shift from 95% branded baseline toward organic discovery.
  • Context: Most keywords started at zero impressions. These are GSC-indexed non-branded queries, not keywords with confirmed search demand.
  • Drove early discovery impressions and clicks across new non-branded queries.
